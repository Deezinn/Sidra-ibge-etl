# Sidra-ibge-etl
SIDRA IBGE ETL Pipeline  Pipeline de ETL em Python para extração, transformação e padronização de dados oficiais do SIDRA (Sistema IBGE de Recuperação Automática) — a plataforma do IBGE que centraliza e disponibiliza dados estatísticos detalhados sobre população, trabalho, economia e outros indicadores socioeconômicos.
